STACK: Python / FastAPI systems
PROJECT: E-Commerce Platform API
DESCRIPTION:
An API for an online store. Users browse product categories, view the products
of a category, and view the full details of a single product.

FEATURES:
CATEGORIES:
- create a category
- view all categories
- view category details
- list products by category

PRODUCTS:
- create a product
- view all products
- view product details

DATA MODELS:
Category:
- name: string
- description: string
- is_active: boolean

Product:
- name: string
- description: string
- price: integer
- image_url: string
- manufacturer: string
- product_dimensions: string
- model_number: string
- country_origin: string
- stock_quantity: integer
- is_available: boolean
- category: Category
