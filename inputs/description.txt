STACK: Python / FastAPI systems

PROJECT: Simple Blog API

DESCRIPTION:
A simple blog API.
It should support users and posts only.

FEATURES:

USERS:
- create user
- get user by id
- list all users
- delete user

POSTS:
- create post (belongs to a user)
- get post by id
- list all posts
- list posts by user
- delete post

DATA MODELS:

User:
- username: string
- email: string

Post:
- title: string
- content: string
- userId: integer
