STACK: Python / FastAPI systems

PROJECT: E-commerce platform API

DESCRIPTION:
We are building an e-commerce platform.
It should support customers, products, orders, and reviews.

FEATURES:

CUSTOMERS:
- register customer
- update customer profile
- delete customer
- get customer by id
- list all customers

PRODUCTS:
- create product
- update product
- delete product
- get product by id
- list all products

ORDERS:
- create order (customer + list of products)
- cancel order
- get order by id
- list orders by customer
- update order status

REVIEWS:
- add review (customer + product)
- delete review
- list reviews by product

DATA MODELS:

Customer:
- name: string
- email: string
- address: string

Product:
- name: string
- price: number
- stock: integer
- description: string

Order:
- customerId: string
- products: list
- status: string
- total: number

Review:
- customerId: string
- productId: string
- rating: integer
- comment: string
